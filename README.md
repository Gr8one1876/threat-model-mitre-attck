# Xibalba Interactive — threat model

**Framework:** MITRE ATT&CK
**Client:** Xibalba Interactive (simulated organization)

---

## Scenario
Xibalba Interactive, a gaming and digital media company, requested a structured threat model to identify the most likely attack vectors targeting their environment and prioritize defensive controls accordingly.

## Objective
Build a comprehensive threat model using the MITRE ATT&CK framework to identify realistic threat actors, map their likely TTPs, and recommend prioritized defensive measures.

## Tools used
- MITRE ATT&CK framework (threat actor and TTP mapping)
- Threat modeling methodology
- Risk scoring and prioritization

## Key findings
- Identified the most likely threat actors targeting the organization's industry and attack surface
- Mapped relevant TTPs across the MITRE ATT&CK matrix
- Prioritized defensive controls based on likelihood and impact

## Outcome
Delivered a structured threat model document covering threat actor profiles, TTP mapping, risk assessment, and a prioritized list of security controls to reduce exposure.

## Skills demonstrated
Threat modeling, MITRE ATT&CK mapping, threat actor profiling, risk assessment, security control prioritization, analytical documentation
